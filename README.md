# Sprint2

This repository contains the CentralPerk loyalty system project for Sprint 2.

## Project Layout

- `LOYALTY_SYSTEM-main/CentralPerk/` — main web application
- `LOYALTY_SYSTEM-main/CentralPerk/supabase/` — SQL scripts, migrations, and Supabase functions

## Stack

- Next.js
- React
- Supabase
- Tailwind CSS

## Run The App

From the app directory:

```bash
cd LOYALTY_SYSTEM-main/CentralPerk
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Notes

- The project was migrated from Vite to Next.js for the frontend framework and build/runtime flow.
- Supabase SQL consolidation and EPIC-LYL-06 campaign/promotion work live under the `supabase/` folder.
